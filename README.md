Titre : Hop (Saut)
# Description: 
ceci un jeu de saut d'un utilisateur sur des barres roulants de taille aléatoire.
# Fonctionnalités
- une interface produite à par de SWING.
- le joueur peut se deplacer en sautant d'une barre à une autre
- les obstacles sur la routes peut reduire la vitesse du joueur
- les picks up permettant de réduire la vitesse des barres
# Utilisation:
1) lancer le fichier /scr/Game.java
2) appuyer sur un boutton pour démarrer
3) les flêches du clavier servent de directions pour le joueur
# structure du projet:
Le projet est réparti en trois packages.
- src
- test
- Vus
